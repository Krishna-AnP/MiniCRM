Mini CRM Application

A Mini CRM (Customer Relationship Management) web application built using :
Node.js, Express, EJS, and SQLite.

The application allows managing Clients, Tasks, and Invoices with a simple
login system and a Dashboard overview.

Tech Stack
- Node.js
- Express.js
- EJS (View Engine)
- SQLite (Database)
- express-session (Authentication)
- bcrypt (Password Encryption)

Prerequisites
Make sure you have the following installed:
- Node.js (v16 or above)
- npm
